# PocketChange ($PCP) Protocol - Milestone 3

## Vision
To build the first community-owned arbitrage protocol on Solana. By pooling user funds and leveraging advanced execution strategies, $PCP democratizes access to high-frequency arbitrage profits.

## Target Audience
Retail crypto investors seeking institutional-grade yield via arbitrage, without the infrastructure complexity.

## Core Value Proposition
- Turn pocket change into wealth through pooled high-frequency arbitrage.
- $PCP token's value is dynamically inflated using 80% of auto-compounded arbitrage profits.
- Non-custodial operations using atomic Programmable Transaction Blocks (PTB).

## Technical Foundation (Milestone 3)
- **Devnet Deployment**: Deploy the smart contract to Solana Devnet for real-world testing.
- **PostgreSQL Telemetry**: Enable scalable telemetry and audit logging of all executed arbitrage routes.
- **Full End-to-End Simulation**: Users can deposit Devnet USDC, receive $PCP, and the bot actively trades on Devnet DEXs (or mocks realistic trades against devnet state) while logging to DB.
