# Twitter/X Post Directive — PCP

## Purpose
Auto-post live arb engine stats to @PocketChangePCP on X/Twitter.

## Schedule
- **Routine recap**: every 6 hours (00:00, 06:00, 12:00, 18:00 UTC)
- **Profit milestone**: any time cumulative session profit crosses 0.1 SOL increments
- **Engine start**: once on boot

## Stats to include (pulled from trade_log.jsonl)
| Field | Source |
|-------|--------|
| Trades today | count of records with today's date |
| Profit (SOL) | sum of `netProfitSol` |
| Profit (USD) | profit × current SOL price from price_feed |
| Best trade | max single `netProfitSol` |
| Top token | most frequent profitable `tokenSymbol` |

## Tone guidelines
- Native Crypto Twitter (CT) voice — bullish, punchy, emoji-forward
- Max 240 chars
- Always include CA: `4yfwG2VqohXCMpX7SKz3uy7CKzujL4SkhjJMkgKvBAGS`
- End with `#Solana #DeFi #NFA`
- Copy generated by: `gemini -p "[prompt]"` (subprocess call, free)

## Edge cases
- If trade_log is empty / no trades today → use simulated stats, add "[SIM]" to tweet
- If Twitter API returns 429 → backoff 15 min, retry once, then skip
- If gemini copy gen fails → use fallback template string

## Fallback template
```
⚡ PocketChange Arb Engine Live
Trades: {trades} | Profit: {profit_sol} SOL (${profit_usd})
Best: {best_sol} SOL in one trade 🔥
CA: {CA}
#Solana #DeFi #NFA
```
